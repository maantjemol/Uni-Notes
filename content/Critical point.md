## Definition
if $f''(x)>0$ on $I = f$ is _concave up_
if $f''(x)<0$ on $I = f$ is _concave down_
if $x_{0}$ is an _inflextion point_ and $f''(x)$ EXISTS, $f''(x)=0$
![[Pasted image 20221014092526.png]]
## Example:
$$\begin{align} 
	f(x) &= x(2x)e^{-x} \\
	f''(x)&=(2-2x)e^{-x}+(2x-x^{2})e^{-x}\cdot(-1)\\
	&=[2-4x+x^2]e^{-x}\\
	\\
	f''(0) = 2 > 0\\
	f''(2) < 0
\end{align}
$$
